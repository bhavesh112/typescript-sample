# typescript-sample
A sample google maps project using typescript.

# Run Locally

Clone the repo

```bash
git clone https://github.com/bhavesh112/typescript-sample
```

Goto the root directory

```bash
cd typescript-sample
```

Install Parcel

```bash
npm i -g parcel
```

Install Dependencies

```bash
npm i 
```

Start the application

```bash
parcel index.html
```
